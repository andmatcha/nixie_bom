## 2. 結論サマリ

- 採用可否: 条件付きで採用可能
- 主な理由: 12 V入力は推奨VDD範囲内で、外付けMOSFET方式の昇圧構成に適合する。170 V出力はIC端子に直接印加せず、外付けMOSFET、ダイオード、出力コンデンサ、帰還抵抗の定格で成立させる設計であれば使用できる。
- 設計上もっとも重要な制約: 高昇圧比による高デューティ動作、最小オフ時間、電流検出抵抗と内部スロープ補償の上限、起動時の出力コンデンサ充電電流、標準昇圧回路では出力短絡時に入力から出力へ電流経路が残ること。
- 実機で必ず確認する項目: 10.8 V入力時の起動、170 V / 20 mA負荷時のSW波形、ISNS波形、出力リップル、COMP安定性、起動時過電流、CAN途絶時DIS/EN停止、電源断後の放電、MOSFET/インダクタ/ダイオード温度。
- 未確認リスク: 外付け部品それぞれの詳細データシートに基づく熱、飽和、逆回復、DCバイアス、抵抗電圧定格は別途確認が必要。

## 4. カテゴリ別確認事項

### 4.1 電源IC・DC/DCコントローラ

- 入力電圧範囲: データシート推奨はVDD 4.5 Vから52 V。今回の12 V nominal、10.8 Vから13.2 V確認範囲は適合する。
- FB基準電圧: TPS40210は700 mV typ、全温度範囲で686 mVから714 mV。
- スイッチング周波数: 35 kHzから1000 kHz。外付けRT/CTで設定する。
- 最小オン/オフ時間: VDD=12 Vで最小オン時間400 ns max、最小オフ時間200 ns max。
- 電流検出: ISNSが過電流しきい値を超えるとヒカップ動作に入り、SSコンデンサで再試行時間が決まる。
- ゲートドライブ: 内蔵400 mA級低側ゲートドライバ。Nch MOSFETのソースをGND基準で使う構成に適合する。
- BPレギュレータ: 約8 V出力。1 uFバイパスが必須。外部負荷を接続する場合はIC損失を確認する。
- DIS/EN: 高レベルでシャットダウン、低レベルまたは未接続で有効。シャットダウン時はBPレギュレータも停止する。
- レイアウト: VDD/BP/ISNS/RC/FB/COMP/SS周辺をIC近傍に置き、SWノードと大電流ループを小さくする。
- 異常時: 標準昇圧トポロジでは出力短絡時に入力からインダクタとダイオード経由で出力へ電流が流れるため、必要に応じてヒューズなど二次保護が必要。

## 7. 設計式と今回条件への適用

### 7.1 デューティ比と最小オフ時間

- 式: CCMでは D = 1 - VIN / (VOUT + VD)
- データシート上の制約: TPS40210の最小オフ時間は200 ns max、最小オン時間は400 ns max。
- 今回の設計条件: VIN = 10.8 Vから13.2 V、VOUT = 170 V、VD = 1 V、fSW = 約167.9 kHz。
- 計算結果:

| VIN | Duty | tON | tOFF |
| ---: | ---: | ---: | ---: |
| 10.8 V | 93.68 % | 5.58 us | 376 ns |
| 12.0 V | 92.98 % | 5.54 us | 418 ns |
| 13.2 V | 92.28 % | 5.50 us | 460 ns |

- 設計上の注意: 最悪側の10.8 V入力でもtOFFは200 ns maxを上回る。周波数を大きく上げると余裕が減る。

### 7.2 発振周波数

- 式: データシートEq.14でRTをfSWとCTから求める。
- 変数: RTはkOhm、fSWはkHz、CTはpF。
- 今回の設計条件: CRC = 100 pF、RRC = 1.00 MOhm。
- 計算結果: fSW = 約167.9 kHz。
- 設計上の注意: RRCはデータシート推奨範囲の上限。許容差と温度で周波数は変動するが、最小オフ時間にはまだ余裕がある。

### 7.3 インダクタ電流

- 式: IRIPPLE = VIN x D / (L x fSW)、ILavg = IOUT / (1 - D)、ILpeak = ILavg + IRIPPLE / 2。
- 今回の設計条件: L = 1 mH、IOUT = 20 mA。
- 計算結果:

| VIN | IRIPPLE | ILavg | ILpeak | ILrms | IOUT(crit) |
| ---: | ---: | ---: | ---: | ---: | ---: |
| 10.8 V | 60 mA p-p | 317 mA | 347 mA | 317 mA | 1.9 mA |
| 12.0 V | 66 mA p-p | 285 mA | 318 mA | 286 mA | 2.3 mA |
| 13.2 V | 73 mA p-p | 259 mA | 295 mA | 260 mA | 2.8 mA |

- 設計上の注意: 20 mA負荷ではCCM。通常動作電流に対してL1の900 mA/1 A級定格は余裕があるが、過電流しきい値まで含めると別途確認が必要。

### 7.4 電流検出抵抗と過電流

- 式: CCMではデータシートEq.16。スロープ補償上限はEq.19。
- データシート上の制約: VISNS(oc) = 120 mV min, 150 mV typ, 180 mV max。高デューティCCMではRISNSをEq.19の80 %以下にすることが開始点として推奨される。
- 今回の設計条件: L1は7447709102を維持し、RSNS = 0.15 Ohm、L = 1 mH、fSW = 約167.9 kHz。
- 計算結果:

| 項目 | 計算値 |
| --- | ---: |
| VIN=10.8 VでのRISNS(max), Eq.19 | 0.189 Ohm |
| 上記の80 % | 0.151 Ohm |
| 採用RSNS | 0.150 Ohm |
| VISNS=120 mV時の概算ピーク検出電流 | 0.80 A |
| VISNS=150 mV時の概算ピーク検出電流 | 1.00 A |
| VISNS=180 mV時の概算ピーク検出電流 | 1.20 A |
| VIN=10.8 V時の出力過電流目安 | 約49 mA / 61 mA / 74 mA |

- 設計上の注意: 0.15 OhmはEq.19の80 %目安にほぼ一致する。通常動作ピーク約0.35 Aに対しては余裕があり、20 mA負荷の起動条件にも余裕を作れる。一方、VISNS最大側ではピーク検出電流が1.2 A相当となり、L1のIsat 1 Aを超え得るため、実機波形と入力側ヒューズの時間電流特性を確認する。

### 7.5 ISNSフィルタ

- 式: tON = D / fSW、RIFLT x CIFLT = 0.1 x tON。
- 今回の設計条件: RIFLT = 1 kOhm、CIFLT = 470 pF。
- 計算結果: tONは約5.5 us、推奨時定数は約0.55 us。採用値は1 kOhm x 470 pF = 0.47 us。
- 設計上の注意: データシート式の目安に近づいたため、100 pF時よりISNSスパイク耐性が上がる。電流制限応答を過度に遅らせないため、SW波形とISNS波形で誤検出と遅延を確認する。

### 7.6 ソフトスタート

- 式: データシートEq.1。VDD > 8 Vの設計例ではCSS = 20 x TSS x 10^-6。
- 今回の設計条件: CSS = 1 uF、BP = 約8 V。
- 計算結果: RSS(chg)の範囲を含めるとtSSは約32 msから60 ms。typ近傍では約43 ms。
- 設計上の注意: COUT = 3 uF nominalを170 Vまで充電する電流は最短tSS側でも約16 mA程度となる。起動中に20 mA負荷も同時にかかる条件では合計約36 mAとなり、RSNS = 0.15 OhmのVIN=10.8 V最小側出力過電流目安約49 mAを下回る。実際はMLCC実効容量、補償応答、負荷の非線形性で変わるため、20 mA負荷付き起動は実測確認する。

### 7.7 出力コンデンサ

- 式: データシート設計例のEq.45では COUT = 8 x IOUT x D / (VOUT(ripple) x fSW)。
- 今回の設計条件: IOUT = 20 mA、Dmax = 93.68 %、fSW = 約167.9 kHz。
- 計算結果: 1 Vリップル目標なら約0.89 uF、0.5 Vリップル目標なら約1.79 uF。
- 設計上の注意: COUT 1 uF / 250 V x 3はnominalでは十分。ただしX7R MLCCは170 V DCバイアスで実効容量が低下するため、実機リップル測定またはMurata特性確認が必要。

### 7.8 入力コンデンサ

- 式: データシート設計例のEq.47では CIN > IRIPPLE / (4 x VIN(ripple) x fSW)。
- 今回の設計条件: IRIPPLE最大約73 mA p-p。
- 計算結果: VINリップル60 mV目標では約1.5 uF以上。
- 採用値: CIN 10 uF x2に加え、入力バルクとしてCINB 22 uF x2を追加する。
- 設計上の注意: 容量計算上はCIN 10 uF x2だけでも足りるが、データシート9章は入力源が数インチ以上離れる場合に追加バルク容量が必要になり得るとしている。ACアダプタ入力と配線インピーダンスを見込み、Murata MLCCの22 uF x2を追加する。実容量はDCバイアスで低下するため、起動時VINリップルを実測する。

### 7.9 FB分圧

- 式: VOUT = VFB x (1 + Rtop / Rbottom)。
- 今回の設計条件: Rtop = 20 kOhm x 5 = 100 kOhm、Rbottom = 412 Ohm。
- 計算結果:

| VFB条件 | VOUT |
| --- | ---: |
| 686 mV min | 167.2 V |
| 700 mV typ | 170.6 V |
| 714 mV max | 174.0 V |

- 設計上の注意: 抵抗許容差を除いた範囲は160 Vから180 Vの調整余地内。上側100 kOhmはデータシート推奨の上限。分圧電流は約1.69 mA、上側抵抗合計の損失は約0.29 W、20 kOhm 1本あたり約57 mW。

### 7.10 補償回路

- 式: データシートEq.22からEq.29で初期値を求め、最終的には周波数応答測定または過渡応答で確認する。
- 今回の設計条件: RCOMP = 68 kOhm、CCOMP = 2200 pF、CHF = 47 pF。
- 計算結果: ゼロは約1.06 kHz、RCOMPとCHFの高周波ポールは約49.8 kHz。
- 設計上の注意: fSWの10 %は約16.8 kHzであり、初期値としては極端ではない。ニキシー管のダイナミック負荷、無負荷、20 mA負荷で安定性を確認する。

### 7.11 MOSFET、ダイオード、ゲート抵抗

- 式: ダイオード耐圧はデータシート設計例のEq.41で VBR >= VOUT / 0.8。ゲート抵抗はEq.30で RG = 105 / QG。
- 今回の設計条件: VOUT = 170 V。
- 計算結果: ダイオードおよびMOSFETの耐圧目安は最低約213 V以上。600 V品のSTD10N60M2とSTTH2R06Uは耐圧面で十分余裕がある。
- 設計上の注意: QG、逆回復、スイッチング損失、発熱は各外付け部品データシートと実測で確認する。

## 8. レイアウト・実装上の注意

- 電源・GND: VDDバイパスはVDDピン近傍。BPの1 uFはBP/GNDピン近傍。
- 大電流ループ: MOSFET、ダイオード、出力コンデンサ、電流検出抵抗を含むループを最小にする。
- 高dv/dtノード: SWノードは物理的に小さくし、寄生容量と放射を抑える。
- アナログ小信号ノード: RC、FB、COMP、SS、ISNSフィルタはTPS40210近傍に集め、SWノードから離す。
- 電流検出: RIFLT/CIFLTはISNS/GNDピン近傍。MOSFETソースからRSNS、GNDピンへの戻り経路の寄生を小さくする。
- ゲート駆動: GDRVトレースはMOSFETゲートにできるだけ近く、戻り経路も短くする。
- 入力: ACアダプタや配線が長い場合、セラミック入力コンデンサに加えてバルク容量を置く。
- 高電圧クリアランス: TPS40210データシート上の範囲外のため、PCB設計規則と安全要件で別途決める。
- テストポイント: VIN、VOUT、SW、ISNS、COMP、DIS/EN、GNDを初回評価で測れるようにする。

## 12. 実機検証チェックリスト

### 共通

- [ ] 10.8 V、12.0 V、13.2 VでVDDが推奨範囲内にある。
- [ ] VDD、RC、SS、FB、DIS/EN、ISNS、COMP、BP、GDRVが絶対最大定格を超えない。
- [ ] U1近傍にVDD/BPバイパスが配置されている。
- [ ] TPS40210、MOSFET、ダイオード、インダクタ、電流検出抵抗の温度が許容範囲内。

### 電源回路

- [ ] 無負荷で170 Vに立ち上がる。
- [ ] ダミーロードで170 V / 20 mAが安定する。
- [ ] 起動時にISNSが過電流しきい値へ達しない。
- [ ] 20 mA負荷を起動時から接続した条件でヒカップしない。
- [ ] 実運用のHV安定後に表示負荷を有効化するシーケンスでも再起動時にソフトスタートする。
- [ ] SWノードのオフ時間が最小オフ時間に対して余裕を持つ。
- [ ] ISNS波形に過大なスパイクがなく、RIFLT/CIFLT値が妥当。
- [ ] 出力リップルが許容範囲内。
- [ ] COMP波形に低周波発振や過大なハンチングがない。
- [ ] DIS/ENで高電圧電源が停止し、再有効化時にソフトスタートする。
- [ ] 電源断後、高電圧が放電抵抗で安全側へ下がる。

### 保護

- [ ] FHV1の通常起動時突入電流と時間電流特性が問題ない。
- [ ] 出力短絡時にFHV1が入力源、L1、D1を許容範囲内に保護できる。
- [ ] L1の飽和電流が通常動作だけでなく過電流しきい値に対しても妥当か確認する。
